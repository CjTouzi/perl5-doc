=EN Directory Handle References
=CN 目录句柄引用
=EN In the same way that we can create references to filehandles, we can create directory handle references. 
=CN 用创建对句柄的引用同样的方法, 我可以创建对目录句柄的引用. 
=EN The directory handle reference obeys the same rules we laid out before. This only works if the scalar variable does not already have a value, and the handle automatically closes when the variable goes out of scope or we assign it a new value. 
=CN 对目录句柄引用遵从我们在之前我们说过的规则. 它必须在标量变量没有值的情况下才能工作, 跑出范围或变量被另外赋值时, 句柄自动关闭. 
=EN We can use object-oriented interfaces for directory handles too. The IO::Dir module has been part of the standard Perl distribution since 5. 6. It doesn't add interesting new features but wraps the Perl built-in functions.
=CN 我们也是用面向对象的接口来处理目录句柄. 从Perl 5. 6开始, IO::Dir模块就是标准发行版的一部份了. 它并没有添加什么新的功能, 无非包装了Perl的内建函数而已. [+]
=EN For each IO::Dir method name, append "dir" and look at the documentation in perlfunc. 
=CN 对于每个IO::Dir模块名, 加上"dir" 并用perlfunc查看其文档.
